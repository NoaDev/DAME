Editing stacked tilemaps in DAME

In DAME you can set up tilemaps to be stacked, allowing you to place tiles in the z axis, when normal maps only deal with x and y.

  1. Setting Up
  2. Editing

1. Setting Up

In order to set up a tilemap to be stacked you first need to enable it in the Layer Properties window for the tilemap. Make sure you click the Has Height checkbox to enable it and specify a Stack Height value. Stack Height controls the number of pixels each tile will be offset as it moves up the stack.

While it is not necessary for stacking, the Base Tile Height value is useful given that most stacked tilemaps tend to be 2.5d tilemaps, i.e. representations of a 3d object in 2d space. This controls the spacing of tiles vertically within each stack, and is relevant even if you are not stacking tiles. By correctly manipulating both this and the Stack Height value you can get a good layout for stacking.

2. Editing

There are two ways to edit stacked tilemaps, with various advantages and disadvantages depending on what you want to do with your tilemap:

  1. Default stacking.
  2. Infinite tile stacking.

1. Default stacking

In the default stacking mode you can only place tiles on top of existing tiles. If you hold down the Z key while placing down tiles you will be able to place a tile on the stack above the current tile. The box indicating the current tile will split to indicate the base tile and the current stacked tile. When it is split any left click or right click will modify that tile rather than the base tile. You can also use the [ and ] keys to move down and up the tile stack respectively. When you right click, if the current tile is the top-most tile in the stack the cursor will move down the stack, otherwise it will stay in place.

2. Infinite tile stacking.

In infinite tile stacking mode you can freely place tiles in the z axis wherever you wish, regardless of whether a tile is below it or not. To enable it go to the Edit Menu and select Allow Infinite Tile Stacking. In this mode, if you move the mouse while holding down Z you will be able to freely slide the current stack location up and down. You can then place down a tile wherever the cursor is. Because infinite tile stacks can be placed as high as you wish, the tile cursor will be located to where the current tile in the stack is rather than the base tile.

An important point to note about stacked tiles is that they are designed to work only when the draw index is greater than 0. In other words there needs to be a blank tile otherwise the tile stacks will not draw or behave correctly.
