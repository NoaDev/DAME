1. The Main Window

Below is a screenshot of DAME's main window along with information about the important areas of the DAME window.

  1. Layers Tab - Click this button to display the Layer pane in the Panel area.
  2. Sprites Tab - Click this button to display the Sprites pane in the Panel area.
  3. Edit Mode Toolbar - Buttons on this toolbar change between the main editing modes.
  4. Current Editor Toolbar - Buttons here change depending on the current edit mode or selected object.
  5. Panel Area - This displays either the Layer pane or the Sprites pane.
  6. Properties Grid - This will display if you are editing a tilemap or an object and displays the current item's properties.
  7. Main Area - This is where the magic happens! The map will show up in here and this is where you place objects.
  8. Status Bar - Some information shows up here such as the current X,Y coordinates and the time since the last save.
  9. Main Menu - Access this to change some of the display/editing features as well as to save, load and export..

2. Navigation

You can navigate around a DAME map many ways:

  • Hold down the middle mouse button in the Main Area and drag the view around.
  • Hold down both the 'D' key and the middle mouse button in the Main Area to drag the view around at a constant rate determined by the distance of the mouse from the initial drag area.
  • Use Ctrl with the Arrow keys to move the view around.
  • F2 to move to (and select) the previous object to the left/up (if valid) on the current layer.
  • F3 to move to (and select) the next object to the right/down (if valid) on the current layer.
  • Ctrl+L (or Main Menu->View->Center Layer) to move the view to the center of the current layer.
  • Scroll the mouse wheel to zoom the display in and out.

3. Transformations

There are 3 types of transformation supported by DAME:

  1. Movement - Tiles and all objects support this.
  2. Rotation - Only objects support this.
  3. Scaling - Only objects support this.

It is only possible to transform an object when it has the dotted lines around it. This goes for tilemaps as well. Usually you initiate transformation mode by selecting the object/tile when in selection mode. This can be done by clicking on the selection icon: . With objects it is also possible to transform the last placed down object.

With the object selected you can move it by clicking anywhere on the object with the left mouse button and dragging it around the map.

Scaling can be done by doing the same as the above but holding down the Alt key. Rotation can be done by doing the same again, but holding down the R key instead.

An alternative way of scaling is to click on the transformation handles that appear around the edges of an already selected object when you move the cursor over it. You will see the cursor icon change to indicate scaling is possible Likewise, if you move the cursor over the edges of the object the cursor icon will change to indicate rotation is possible. Clicking the left mouse button at this time will begin a rotation.

You can constrain rotation to within 45 degree angles by holding down the 'A' Key. To constrain scaling so that it is uniform only hold down the 'S' Key.

4. Grids and constraints

You can display the grids by going to the menu "View->Grid" (Ctrl+G). If you wish to change the size of the guidelines this can be done two ways:

  1. As guidelines will always be aligned to the tilemap when you are on a tilemap layer you can go to the menu Edit->Guides->Set From Current Layer. This will ensure that whatever layer you are on the guidelines will be aligned to this layer.
  2. Go to "View->Change Grid Settings..." to bring up a window that will let you select all of the gridline parameters. For most settings this is only relevant to non tilemap layers. By default tilemaps will always display lines around every tile, though you can stagger the lines using the "Min Grid Space" value. This controls the minimum spacing between any two grid lines. The "Snap Location" value controls the location of the snapping point of an object when it is snapped to a grid.

Displaying the grid doesn't necessarily mean that objects will be snapped into place when you move/add them. To do this select the menu option "View->Snap To Grid" (Ctrl+Shift+G).

If you have a tilemap and want to quickly align non-tilemap layers to the tilemap then you can select the menu option "View->Align Grids To Current Layer".

An alternative method of constraining movement/placement is to hold down the 'X' key to ensure that movement of the cursor will only occur horizontally, and the 'Y' key (or 'Z' key) to ensure it can only occur vertically.

One other feature available in the grid settings is the Game Region Overlay. This is also available through the menu option "Tools->Show Game Region Overlay" (Ctrl+Y). When active it will display opaque or partially transparent over the main view of the editor, leaving only the region specified in the settings clear. This will allow you to see how your game might look in game.
